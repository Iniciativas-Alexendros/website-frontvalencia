Initial release of FRONT Valencia website.

- Astro 5 SSG with 21 bilingual pages (ES/EN)
- Payload CMS with collections (Menu, Space, Events, Allergens)
- RBAC (admin/editor/viewer)
- i18n with hreflang and sitemap
- Docker Compose for local development
- Seed data (14 allergens, 8 categories, 44 dishes)
- JSON-LD structured data (Restaurant, WebSite, BreadcrumbList)
- SEO: sitemap, hreflang, Open Graph, Twitter Cards, performance budget
- Accessibility: axe-core checks, skip links, keyboard navigation, reduced motion
- CI/CD: 5 GitHub Actions workflows
- Quality: ESLint, Prettier, lint-staged, commitlint, Vitest, Playwright
